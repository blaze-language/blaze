## 🏁 Getting Started

### 📋 Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [GCC](https://gcc.gnu.org/install/)

### 🔧 Installation

```console
$ git clone https://github.com/blaze-lang/blaze
$ cd blaze
$ ./install.sh
```

## 🔨 Usage

```console
$ blaze run <file>
```
